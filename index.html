¿Qué estamos buscando?

Nos encontramos ante la tarea de encontrar un MiniPC que pueda realizar funciones de servidor.

Estas funciones son excluyentes, si realiza una de ellas, no tendrá que realizar las demás. La lista de tareas que debe poder realizar es la siguiente:

  • Firewall
  • HTPC
  • Steam Link
  • Servidor de ficheros
  • Servidor DNS
  • Servidor de VPN
  • Servidor de impresión

¿Qué sistema operativo vamos a ejecutar?

Podremos escoger un sistema operativo de esta lista, que se encuentra en orden de prioridad descendente:

  1. GNU/Linux: Debian
  2. GNU/Linux: Ubuntu
  3. FREEBSD pfSense
  4. Windows Server 2019
  5. Mac OSX BigSur

Ya que Debian es el favorito, podremos ver aquí su documentación sobre Debian.
